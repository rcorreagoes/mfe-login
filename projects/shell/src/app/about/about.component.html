Sobre: Micro Frontend Angular

Micro Frontends é um estilo arquitetural onde uma aplicação frontend é dividida em "microaplicações" individuais e semi-independentes, que trabalham juntas de forma integrada. No ecossistema Angular, isso geralmente é feito usando Module Federation, permitindo que múltiplas aplicações Angular sejam desenvolvidas, implantadas e atualizadas de forma independente, mas carregadas juntas em tempo de execução para uma experiência unificada.

Este projeto demonstra uma arquitetura de micro frontends com Angular. Ele é composto por um micro frontend de login (remoto) e uma aplicação shell (host). O MFE de login fornece telas de autenticação como login, esqueci minha senha e alteração de senha, todas estilizadas com Angular Material. A aplicação shell carrega e exibe o MFE de login dinamicamente, permitindo desenvolvimento e deploy independentes.

  • Cada micro frontend pode ser desenvolvido e implantado separadamente.
  • Module Federation permite compartilhar código e funcionalidades entre apps em tempo de execução.
  • Essa abordagem melhora a escalabilidade, manutenção e autonomia dos times.

Funcionalidades implementadas neste projeto:

  • Tela de login com Angular Material
  • Fluxo de recuperação de senha com validação de email e código
  • Tela de alteração de senha com confirmação
  • Design moderno e responsivo
  • Integração micro frontend usando Module Federation
